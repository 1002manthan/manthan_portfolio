## ✨ Overview

A **fully responsive personal portfolio** built with **Flutter** — runs as a **Web app** and an **Android app** from a single codebase. Showcases my projects, skills, internship experience, and education with smooth scrolling, dark/light mode, and an animated orbiting skill system.

🌐 **Web:** [1002manthan.github.io/manthan_portfolio](https://1002manthan.github.io/manthan_portfolio/)  
📱 **Android:** Download APK Build from my GitHub

## 🚀 Features

| Feature | Description |
|---|---|
| 🪐 **Orbiting Skills** | Animated dual-ring orbit — Flutter, Dart, Java & more rotating around a glowing core |
| 🌙 **Dark / Light Mode** | Animated toggle that transitions every color across the entire UI |
| 🌊 **Smooth Scrolling** | Custom scroll engine — silky mouse-wheel on web, natural touch fling on mobile & Android |
| 📱 **Fully Responsive** | Mobile (`<600px`), Tablet (`600–1023px`), Desktop (`≥1024px`) — layouts, fonts & spacing all adapt |
| ☰ **Mobile Nav Drawer** | Bottom-sheet hamburger menu with smooth dismiss + scroll animation |
| ⚡ **Section Animations** | Fade + slide reveal on every section as you scroll |
| 🎯 **Animated Skill Bars** | Progress bars animate in on first view with `easeOutCubic` curve |

## ▶️ Run Locally

### Web
```bash
flutter pub get
flutter run -d chrome
```

### Android
```bash
flutter pub get
flutter run                    # runs on connected device / emulator
```

### Build APK
```bash
flutter build apk --release
# Output: build/app/outputs/flutter-apk/app-release.apk
```

### Build Web
```bash
flutter build web --release --web-renderer canvaskit --base-href "/manthan_portfolio/"
```

## 📁 Project Structure

```
lib/
├── main.dart                    # App entry + theme state
├── portfolio_page.dart          # Scroll controller + layout scaffold
├── theme/
│   ├── app_theme.dart           # AppColors (light/dark) + InheritedWidget
│   └── responsive.dart          # Breakpoints + R.hPad() / R.vPad() helpers
├── widgets/
│   ├── nav_bar.dart             # Sticky nav + hamburger bottom-sheet drawer
│   ├── orbit_widget.dart        # Animated dual-ring orbiting skills
│   └── common_widgets.dart      # SectionHeader, AnimatedReveal
└── sections/
    ├── hero_section.dart
    ├── about_section.dart
    ├── skills_section.dart
    ├── projects_section.dart
    ├── experience_section.dart
    ├── education_section.dart
    └── contact_section.dart
```
